# Task 5: Laporan dan Dasbor (Reporting & Dashboard)

## Objektif
Menyediakan fitur dasbor analitik real-time dan laporan eksportir (PDF/Excel) berdasarkan data anggota, simpanan, pinjaman, dan angsuran.

## Detail Pekerjaan
1. **Dasbor Analitik (Dashboard)**
   - Buat halaman ringkasan dasbor.
   - Tampilkan visualisasi metrik penting: total anggota aktif, total simpanan terkumpul, total pinjaman berjalan, arus kas (cash flow) menggunakan data dummy hasil seeding di Task 1-4.

2. **Modul Laporan**
   - Implementasikan fitur generate Laporan Keuangan sederhana.
   - Integrasikan package atau fungsi export data ke format PDF atau Excel.
   
3. **Kriteria Selesai (Acceptance Criteria)**
   - Dasbor menampilkan data metrik yang benar sesuai data seeder di dalam database.
   - Fitur export laporan ke PDF/Excel dapat diunduh tanpa error.
